pymunk-edu by Darion

Лог изучения pymunk через разработку

Итак, преодолев стеcнение, я создал публичный клон репозитория для своих ковыряний в pymunk. Здесь будут логи "разработки" для структурирования процесса изучения и экспериментирования.

По сути, скорее всего это никому кроме меня не будет нужно. Но раз github создает удобные инструменты для работы с проектом, почему бы ими не воспользоваться.

Логов пока нет, я не разобрался в том, как их удобнее всего будет вести. Но скоро будут.

Download

Если это вам зачем-то понадобилось, вы можете скачать исходники в форматах zip или tar.

А также клонировать проект через Git командой: 

$ git clone git://github.com/Darion/pymunk-edu
и всласть посмеяться. Ну или поругаться на "криворуких python-программистов".
